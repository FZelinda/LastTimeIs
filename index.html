Privacy Policy

Effective Date: January 18, 2025

Data Collection

We commit to not collecting any user data, including but not limited to:

  • Personal identity information (name, email, phone, etc.)
  • Device identifiers (IDFA, device model, etc.)
  • Usage behavior data (taps, views, etc.)
  • Location information
  • Usage analytics and statistics

Data Storage

All app data is stored locally on your device using the iOS SwiftData framework. Stored data includes:

  • Event records (name, type, icon, color)
  • Log entries (timestamp, notes, photos)
  • Category settings
  • App preferences

You can export a backup at any time using the "Export Data" feature, or delete all local data using the "Clear Data" feature.

Permissions

The app only requests permissions when you actively use the following features:

  • Photo Library Access: Only when selecting an existing photo as a log attachment
  • Camera Access: Only when taking a new photo as a log attachment

All permissions are fully under your control. The app will not access the photo library or camera in the background.

Third-Party Services

This app does not integrate any third-party SDKs, including but not limited to:

  • Analytics tools (such as Google Analytics, Firebase)
  • Advertising networks
  • Social sharing SDKs
  • Push notification services

Children's Privacy

This app is not designed for children under 13 and does not knowingly collect any personal information from children. Since we collect no data, there are no privacy risks related to children.

Changes to Privacy Policy

If we update our privacy policy, we will note the changes within the app and in new version descriptions. Significant changes will be communicated through app update notifications.
